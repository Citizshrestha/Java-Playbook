JDBC — Steps to Execute SQL Queries

JDBC (Java Database Connectivity) is a Java API that allows Java applications to interact with relational databases.

  • It enables developers to execute SQL queries, retrieve results, and perform database operations such as INSERT, UPDATE, DELETE, and SELECT.
  • JDBC is part of the java.sql package and works with different databases such as MySQL, Oracle, and PostgreSQL.

Steps to Execute SQL Queries Using JDBC

1
Import Packages

Import the required Java SQL packages before using any JDBC classes.

import java.sql.*;
2
Load the JDBC Driver

Load the database driver class into memory so Java knows how to communicate with the database.

Class.forName("com.mysql.cj.jdbc.Driver");
3
Establish a Connection

Create a connection to the database using the URL, username, and password.

Connection conn = DriverManager.getConnection(url, user, pass);
4
Create a Statement

Create a Statement or PreparedStatement object used to send SQL queries to the database.

Statement PreparedStatement
When to use
Static SQL with no user input or parameters.		 When to use
Dynamic SQL with parameters (user input, form data, etc.).
Query is compiled every time it runs. Query is pre-compiled once and reused — faster for repeated execution.
Vulnerable to SQL injection if values are concatenated directly. Uses ? placeholders — prevents SQL injection automatically.
Statement st = conn.createStatement(); PreparedStatement ps = conn.prepareStatement(query);
5
Execute the Query

Execute the SQL query and store the returned result in a ResultSet object.

ResultSet rs = st.executeQuery(query);
6
Process the Results

Iterate through the ResultSet to retrieve and display the data row by row.

while (rs.next()) { System.out.println(rs.getInt(1) + " " + rs.getString(2)); }
7
Close the Connection

Always close all resources (ResultSet, Statement, Connection) to release database resources and avoid memory leaks.

conn.close();

Example Program

Example: Insert a Customer Record and Display All Customers Aged Above 45
This program inserts a customer into the customer table with fields (cust_id, name, age, address) and then displays all records where age > 45.
Table Structure
CREATE TABLE customer ( cust_id INT PRIMARY KEY, name VARCHAR(60), age INT, address VARCHAR(60) );
Java Code
Inserts one customer record, then queries all customers where age > 45.
import java.sql.*; public class CrudCustomer { public static void main(String[] args) { String url = "jdbc:mysql://localhost:3307/customerDb"; String user = "root"; String pass = "pass123"; try { Class.forName("com.mysql.cj.jdbc.Driver"); Connection conn = DriverManager.getConnection(url, user, pass); // Insert a new customer String query = "INSERT INTO customer(cust_id, name, age, address) VALUES(?, ?, ?, ?)"; PreparedStatement pstmt = conn.prepareStatement(query); pstmt.setInt(1, 101); pstmt.setString(2, "Citiz"); pstmt.setInt(3, 21); pstmt.setString(4, "Gaindakot-2"); pstmt.executeUpdate(); System.out.println("Customer inserted successfully."); // Retrieve customers with age > 45 String query1 = "SELECT * FROM customer WHERE age > 45"; PreparedStatement pstmt1 = conn.prepareStatement(query1); ResultSet rs = pstmt1.executeQuery(); System.out.println("Customers with age > 45:"); while (rs.next()) { System.out.println( "ID: " + rs.getInt("cust_id") + " " + "Name: " + rs.getString("name") + " " + "Age: " + rs.getInt("age") + " " + "Address: " + rs.getString("address") ); } // Close all resources rs.close(); pstmt.close(); pstmt1.close(); conn.close(); } catch (Exception e) { e.printStackTrace(); } } }
