🏛️ JDBC VS 🌿 Hibernate

Comparison Criteria 🔧 JDBC 🌿 Hibernate
Definition A Java API for database connectivity using SQL queries directly An Object Relational Mapping (ORM) framework to map Java objects to database tables
Approach SQL-Centric: Developers write SQL queries manually Object-Centric: Works with Java objects, generates SQL queries automatically
Database Dependency Database-specific SQL code needed for different databases Database-independent (dialect configurable)
Code Length More boilerplate code required Less code, more configuration-based
Performance Faster for simple operations and direct SQL control Optimized with caching mechanisms, slower for simple queries
Learning Curve Easier to learn basics, familiar SQL concepts Steeper learning curve, requires understanding of ORM concepts
Best Suited For Simple operations, batch processing, applications requiring full SQL control Complex domain models, rapid development, long-term maintainability
🔧 JDBC Data Insertion Example
public class InsertData { public static void main(String[] args) { String url = "jdbc:mysql://localhost:3307/empDb"; String user = "root"; String pass = "pass12"; try { // Load MySQL JDBC Driver Class.forName("com.mysql.cj.jdbc.Driver"); // Establish database connection Connection conn = DriverManager.getConnection(url, user, pass); // Prepare SQL query with placeholders String sqlQuery = "INSERT INTO Students(id, name, salary) VALUES(?,?,?)"; PreparedStatement pstmt = conn.prepareStatement(sqlQuery); // Set parameters pstmt.setInt(1, 1); pstmt.setString(2, "Citiz"); pstmt.setInt(3, 200000); // Execute the update int rowsUpdated = pstmt.executeUpdate(); if (rowsUpdated > 0) { System.out.println("Data inserted: rows affected = " + rowsUpdated); } // Close resources pstmt.close(); conn.close(); } catch (Exception e) { e.printStackTrace(); } } }
🌿 Hibernate Data Insertion Example
// Entity class mapping @Entity @Table(name = "employees") public class Employee { @Id @GeneratedValue(strategy = GenerationType.IDENTITY) private Long id; private String name; private Integer salary; // Constructors, getters, and setters... public Employee() {} public void setName(String name) { this.name = name; } public void setSalary(Integer salary) { this.salary = salary; } } // Hibernate session usage public class HibernateExample { public static void main(String[] args) { // Get Hibernate session Session session = HibernateUtil.getSessionFactory().openSession(); Transaction tx = session.beginTransaction(); // Create and populate employee object Employee emp = new Employee(); emp.setName("Sycon"); emp.setSalary(50000); // Save object (Hibernate generates SQL automatically) session.save(emp); // Commit transaction and close session tx.commit(); session.close(); } }
